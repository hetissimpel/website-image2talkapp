image2talk uses real images, making it suitable for adults and children

Overview screen

  • Simple overview screen appears when icon is tapped
  • Basic rows (yes/no, food, objects, places, clothes, etc) are already in image2talk
  • Choose an image - it appears and speaks
  • Display any number of rows
  • Display any number of images

Define settings for each row

  • Set time delay for basic overview screen to reappear to meet individual needs
  • Choose to show text labels over the image
  • Choose to show text labels only
  • Turn voice-over on/off

Editing rows and images

  • Change the order of rows
  • Change the order of images
  • Delete rows
  • Delete images
  • Edit display text and speaker output

Add your own images and rows

  • Add images from the iPad's camera roll
  • Add a new blank row and fill with images from the iPad's camera roll
  • Save an image from the internet or your camera and import from Dropbox

Using image2talk

  • Use for independent communication across all environments
  • Use for prompted communication across all environments e.g. 'Do you want tea or coffee?'
  • Easily customise image2talk at source to meet individual needs
